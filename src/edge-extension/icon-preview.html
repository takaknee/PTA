📋 使用箇所と設定状況

✅ ツールバーアイコン (action.default_icon)

Edgeのツールバーに表示される拡張機能のメインアイコン。クリックでポップアップが開きます。

✅ 拡張機能アイコン (icons)

拡張機能管理画面（edge://extensions/）で表示されるアイコン。

✅ ストアアイコン (icons)

Edgeアドオンストアでの表示用アイコン（将来的な公開時）。

✅ 設定ページアイコン

オプション画面やその他のUI要素で使用されるアイコン。

manifest.json 設定内容

"action": { "default_popup": "popup/popup.html", "default_title": "AI業務支援ツール", "default_icon": { "16": "assets/icons/icon16.png", "32": "assets/icons/icon32.png", "48": "assets/icons/icon48.png", "128": "assets/icons/icon128.png" } }, "icons": { "16": "assets/icons/icon16.png", "32": "assets/icons/icon32.png", "48": "assets/icons/icon48.png", "128": "assets/icons/icon128.png" }

🚀 次のステップ

1. Edgeで edge://extensions/ を開く
2. 「開発者モード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダ（src/edge-extension）を選択
5. ツールバーにロボット軍曹が表示されることを確認！
